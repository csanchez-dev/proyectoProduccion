# CONIITI 2026 – Propuesta de Frontend

Este proyecto corresponde a una nueva propuesta de rediseño para la página oficial del Congreso Internacional de Innovación y Tendencias en Ingeniería (CONIITI).

El objetivo principal es modernizar la plataforma tanto en su estructura visual (UI/UX) como en su arquitectura técnica, implementando tecnologías actuales que mejoren el rendimiento, la escalabilidad y la experiencia del usuario.

La propuesta integra un frontend dinámico desarrollado en React con TypeScript, manteniendo una estructura organizada basada en HTML semántico.

# Objetivo General

Modernizar la plataforma web del CONIITI mediante:

Mejora de la experiencia de usuario (UX).

Diseño visual actualizado y responsive.

Arquitectura escalable basada en componentes.

Integración eficiente entre frontend y backend.

Código tipado y estructurado utilizando TypeScript.

# 1. Requisitos Previos

asegurate de tener instalado Node.js (v16+) en tu sistema.

Ejecución del Frontend (React + TypeScript)

Ubícate en la carpeta del frontend y ejecuta:

 -Instalar dependencias

npm install

 -Ejecutar servidor de desarrollo

npm run dev

Luego abre en tu navegador:

http://localhost:5173
